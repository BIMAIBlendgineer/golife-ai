# 27 - Analytics Event Taxonomy

## Core events

- `onboarding_started`
- `onboarding_completed`
- `life_event_created`
- `daily_plan_requested`
- `daily_plan_generated`
- `mission_marked_useful`
- `mission_completed`
- `recommendation_rejected`
- `paywall_viewed`
- `subscription_started`

## Recommended properties

- `domain`
- `source`
- `ai_enabled`
- `used_mock`
- `used_fallback`
- `mission_count`
- `blocked_count`
- `time_to_first_mission_seconds`

## Privacy rule

Never log raw personal content, raw prompts, full pantry items, full task text or identifiable notes by default.
