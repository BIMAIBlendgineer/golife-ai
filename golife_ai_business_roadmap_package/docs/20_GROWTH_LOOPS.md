# 20 — Growth Loops

## Loop 1 — Misiones diarias

1. Usuario recibe misión.
2. Completa misión.
3. Ve progreso.
4. Quiere repetir mañana.

## Loop 2 — Ahorro visible

1. App detecta microgasto.
2. Usuario evita compra.
3. App estima ahorro.
4. Usuario comparte resultado.

## Loop 3 — Anti-desperdicio

1. App alerta alimento próximo a vencer.
2. Usuario cocina.
3. App calcula alimento salvado.
4. Usuario confía más.

## Loop 4 — Familia

1. Usuario crea lista/despensa.
2. Invita familiar.
3. Familiar añade compra/tarea.
4. El plan familiar mejora.

## Loop 5 — Outfit/no-buy

1. Usuario evita compra duplicada.
2. App muestra alternativas en armario.
3. Usuario siente valor económico.
4. Registra más prendas.
