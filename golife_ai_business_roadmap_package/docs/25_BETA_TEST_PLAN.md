# Beta Test Plan

## Participants

30 initial testers.

## Duration

14 days.

## Tasks

- complete onboarding;
- add 10 events;
- generate daily plan 5 times;
- mark recommendation feedback;
- answer final survey.

## Survey questions

- Did GoLife reduce daily friction?
- Were missions realistic?
- Which module was most useful?
- Would you pay?
- What felt invasive?
