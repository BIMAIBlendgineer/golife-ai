# 14 - Metrics and KPIs

## North Star

**Useful Missions Completed per Active User per Week**

Definition:

Useful Missions = missions completed and/or explicitly marked useful.

## Product KPIs

- onboarding activation rate
- time to first mission
- D1 retention
- D7 retention
- D30 retention
- weekly active users
- mission completion rate
- recommendation usefulness rate
- capture events per active user

## AI KPIs

- structured output success rate
- fallback rate
- average latency
- safety intervention rate
- schema validation failure rate
- user rejection rate
- AI cost per active user

## Business KPIs

- free to paid conversion
- monthly churn
- ARPU
- gross margin after AI costs
- support burden per paid user

## Guardrail metrics

- generic mission complaint rate
- privacy concern rate
- explanation view rate
- mission repeatability score
