# DDC — Misiones accionables

## Contratos mínimos
- acción sobre tarea
- acción sobre hábito
- acción sobre pantry
- acción sobre closet
- feedback implícito
- avoid repetition

## Reglas
- IDs estables.
- Fechas ISO-8601.
- Payload mínimo.
- Trace para IA.
- Metadata operacional sin contenido sensible.

## Ejemplo
```json
{
  "id": "example",
  "user_id": "local-user",
  "created_at": "2026-04-25T10:00:00Z"
}
```
