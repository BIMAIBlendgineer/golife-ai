# PDR — FridgeZero + Recipe Rescue

## Objetivo
Implementar FridgeZero + Recipe Rescue como una etapa verificable del producto GoLife AI.

## Problema
El usuario necesita que esta capacidad reduzca fricción, mejore claridad y alimente LifeGraph.

## Requisitos funcionales
- CRUD pantry
- cantidad
- vencimiento
- ubicación
- lista compra
- receta rápida
- marcar usado
- comida rescatada

## Requisitos no funcionales
- Privacidad explícita.
- Tests.
- Telemetría cuando aplique.
- Fallback.
- Documentación.
- No introducir payload sensible innecesario.

## Criterios de aceptación
- Flujo funcional de extremo a extremo.
- Tests verdes.
- UI usable.
- Contratos documentados.
- No rompe CI.
