# ADR-008 — No price/sustainability claim without source

## Estado

Aprobado.

## Decisión

EcoShop no puede afirmar:

- mejor precio
- más sostenible
- disponible
- certificado
- recomendado objetivamente

sin fuente, fecha y nivel de confianza.

## Regla

Si falta evidencia:

```text
evidence_status = insufficient_verified_data
```

## Bloqueador

Cualquier claim fuerte sin fuente bloquea release.
