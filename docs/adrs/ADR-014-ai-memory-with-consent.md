# ADR-014 — Memoria IA con consentimiento

## Estado
Aceptado.

## Contexto
La potencia de GoLife está en memoria, pero memoria personal es sensible.

## Decisión
Separar:

- memoria operativa de sesión;
- resumen semanal;
- memoria persistente;
- datos excluidos.

## Consecuencias
- Mejor privacidad.
- UX debe explicar qué se guarda.
