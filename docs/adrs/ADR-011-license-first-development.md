# ADR-011 — Desarrollo license-first

## Estado
Aceptado.

## Contexto
Habo, WeekToDo y Flow son GPL-3.0; otros son MIT.

## Decisión
Antes de copiar código, decidir:

- camino A: GoLife GPL-3.0;
- camino B: usar repos GPL solo como referencia conceptual y reescribir desde cero;
- camino C: contactar autores/licencias alternativas.

## Consecuencias
- Evita riesgo legal.
- Puede afectar modelo comercial.
