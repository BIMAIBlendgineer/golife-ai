# ADR-002 — Flutter como shell móvil principal

## Estado
Aceptado.

## Contexto
Habo, Flow, OpenWardrobe y Taskly son Flutter/Dart. Wanna es Expo/React Native y WeekToDo es Vue/Electron.

## Decisión
Usar Flutter como app principal.

## Consecuencias
- Se puede aprovechar más conocimiento de Habo/Flow/OpenWardrobe/Taskly.
- Wanna y WeekToDo deben reescribirse o adaptarse.
- Un solo runtime móvil simplifica el MVP.
