# GoLife AI Support

Last updated: `2026-05-16`

## Public support path

Current public support path:

- issue tracker: `https://github.com/BIMAIBlendgineer/golife-ai/issues`
- repository home: `https://github.com/BIMAIBlendgineer/golife-ai`

## What to include

When filing a support issue, include:

- app area affected
- expected behavior
- observed behavior
- locale in use
- whether the app was in live, degraded, or local source state
- whether the issue concerns export, delete, privacy, or LifeGraph visibility

Do not include raw personal journal text, receipt images, or sensitive private content unless you explicitly intend to share it in a public issue.

## Privacy or safety concerns

For privacy or safety defects, open an issue with the smallest reproducible metadata needed to describe the problem.

## Local controls already available in app

- protected local export
- delete all local data
- clear AI history
- domain-level privacy controls
- event-level privacy controls

## Store review note

This public support page exists so store metadata can point to a stable support URL before a dedicated external support portal is introduced.
