# GoLife Flutter Shell

Manual Flutter shell created from the GoLife AI prompts.

What is included:

- router and adaptive shell navigation
- dashboard
- lifegraph core
- privacy settings
- local storage abstraction
- mock AI gateway client
- own domain models only

What is not included yet:

- Android/iOS/macOS/Linux/Web runner folders generated by `flutter create`
- source-code migration from external repos
- real AI gateway networking

This workspace does not currently have the Flutter SDK installed, so the shell was authored manually and could not be bootstrapped with platform runners or executed locally in this environment.
