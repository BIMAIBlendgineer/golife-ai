# ADR-016 — Adaptadores de dominio antes de pulir UI

## Estado
Aceptado.

## Contexto
La novedad depende de conectar dominios, no de pantallas bonitas.

## Decisión
Primero implementar:

- LifeEvent;
- adapters;
- AI summaries;
- suggestions.

Después UI premium.

## Consecuencias
- MVP puede ser menos visual al principio.
- Núcleo más sólido.
