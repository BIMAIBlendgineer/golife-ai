# ADR-017 — Pantalla de trazabilidad IA

## Estado
Propuesto.

## Contexto
El usuario debe entender por qué recibe una recomendación.

## Decisión
Crear pantalla `AI Trace` con:

- datos usados;
- modelo;
- confianza;
- incertidumbre;
- acción propuesta;
- permisos.

## Consecuencias
- Diferenciación de confianza.
- Más trabajo de diseño.
