# ADR-008 — Confirmación humana antes de acciones externas

## Estado
Aceptado.

## Contexto
La app puede sugerir compras, cambios de plan o eliminación de datos.

## Decisión
Cualquier acción externa requiere confirmación explícita.

## Consecuencias
- Menos automatización total.
- Menor riesgo.
- Mejor control del usuario.
