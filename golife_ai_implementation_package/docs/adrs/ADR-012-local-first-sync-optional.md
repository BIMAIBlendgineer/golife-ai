# ADR-012 — Local-first con sync opcional

## Estado
Aceptado.

## Contexto
WeekToDo y Flow enfatizan privacidad/offline. GoLife maneja datos sensibles.

## Decisión
Guardar lo esencial localmente y sincronizar solo con consentimiento.

## Consecuencias
- Mejor privacidad.
- Más complejidad de sincronización.
- Mejor resiliencia offline.
