# ADR-015 — MVP antes de superapp

## Estado
Aceptado.

## Contexto
Unir seis productos puede volverse inmanejable.

## Decisión
MVP con:

- tareas;
- hábitos;
- gastos manuales;
- despensa manual;
- misión diaria IA;
- explicación.

Armario y listas compartidas entran después.

## Consecuencias
- Lanza antes.
- Menos riesgo.
- Arquitectura preparada para crecer.
